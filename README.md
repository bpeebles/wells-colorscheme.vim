# wells-colorscheme

yellow highlight is for visual mode and search
![screenshot](http://i.imgur.com/GFEjjzi.png)

#Install (pathogen)
```bash
cd ~/.vim/bundle/
git clone https://github.com/wellsjo/wells-colorscheme.vim.git
```

then in vimrc
```bash
colorscheme wells-colors
```
